# KillerEvent - Minecraft Plugin

Плагин для сервера Minecraft (Paper 1.21.4), который запускает случайное событие, где случайно выбранный онлайн-игрок становится целью для остальных игроков.

## Возможности

- **Автоматическое событие каждые 2 дня**: Случайный онлайн-игрок выбирается в качестве цели
- **Длительность события**: 2 часа на убийство цели
- **Трансляция координат**: Координаты цели отправляются в чат каждые 10 минут
- **Система награда**:
  - Если цель выжила: получает награду выжившего
  - Если цель убита: убийца получает награду за убийство
- **Полностью настраиваемая конфигурация**

## Установка

### Требования
- Java 21+
- Maven
- Paper 1.21.4

### Компиляция

```bash
mvn clean package
```

JAR файл будет создан в папке `target/`.

### Размещение на сервере

1. Скопируйте `KillerEvent-1.0.0.jar` в папку `plugins/` вашего сервера
2. Перезагрузите сервер или используйте команду `/reload confirm`
3. Плагин автоматически создаст конфиг-файл

## Конфигурация

Файл конфигурации находится в `plugins/KillerEvent/config.yml`

### Параметры

```yaml
# Интервал между событиями (в часах)
event-interval-hours: 48

# Длительность события (в минутах)
event-duration-minutes: 120

# Интервал трансляции координат (в минутах)
coordinates-broadcast-interval-minutes: 10

# Система наград
rewards:
  survivor-reward:
    command: "give {player} diamond 5"
    message: "§6§lВы выжили! §eПолучена награда!"
  
  killer-reward:
    command: "give {player} diamond 10"
    message: "§6§lВы убили цель! §eПолучена награда!"
```

### Кастомизация команд наград

Вы можете использовать любые команды, поддерживаемые вашим сервером:

```yaml
rewards:
  survivor-reward:
    command: "give {player} diamond 5"
    message: "§6Вы получили 5 алмазов!"
  
  killer-reward:
    command: "give {player} diamond 10"
    message: "§6Вы получили 10 алмазов!"
```

Используйте `{player}` как плейсхолдер для имени игрока.

## Как это работает

1. **Начало события**: Каждые 2 дня (48 часов) автоматически выбирается случайный онлайн-игрок
2. **Объявление**: Все игроки получают сообщение о выбранной цели и времени события
3. **Трансляция**: Каждые 10 минут координаты цели отправляются в чат
4. **Завершение**:
   - Если цель выжила 2 часа: получает награду выжившего
   - Если цель убита: убийца получает награду за убийство

## Цветовые коды

Используются стандартные коды форматирования Minecraft:
- `§c` - красный
- `§e` - желтый
- `§6` - золотой
- `§l` - жирный
- `§r` - сброс форматирования

## Разработка

### Структура проекта

```
src/main/java/com/killer/
├── KillerEventPlugin.java       - Главный класс плагина
├── EventManager.java             - Менеджер событий
└── PlayerDeathListener.java       - Слушатель смертей игроков

src/main/resources/
├── plugin.yml                     - Метаданные плагина
└── config.yml                     - Конфигурационный файл
```

### Классы

- **KillerEventPlugin**: Главный класс, точка входа плагина
- **EventManager**: Управляет логикой события (выбор цели, таймеры, награды)
- **PlayerDeathListener**: Слушатель для обнаружения смертей игроков
