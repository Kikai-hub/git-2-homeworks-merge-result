# Установка и компиляция KillerEvent

## Требования

- Java 21+ (установлен и добавлен в PATH)
- Maven 3.6+ (установлен и добавлен в PATH)

## Проверка установки

```bash
java -version
mvn -version
```

## Компиляция проекта

### Способ 1: Через командную строку

```bash
cd c:\Users\VIPAR\Desktop\dev\events from server\dev1_Killer_event
mvn clean package
```

После успешной компиляции JAR файл будет находиться в папке `target/`:
- `KillerEvent-1.0.0.jar` - готовый плагин

### Способ 2: Через IDE (VS Code, IntelliJ IDEA и т.д.)

1. Откройте проект в IDE
2. IDE может попросить индексировать Maven проект - подтвердите
3. Дождитесь скачивания зависимостей (это может занять 1-2 минуты)
4. Выполните Maven команду `clean package` через интерфейс IDE

## Если вы видите ошибки типа "cannot access Audience"

Это нормально! IDE может показывать ошибки до того, как Maven скачает зависимости. После того как Maven завершит скачивание, ошибки исчезнут.

**Решение:**
1. Выполните `mvn clean install` в терминале
2. Обновите проект в IDE (F5 в VS Code, Ctrl+Shift+O в IntelliJ)

## Установка на сервер

1. Скопируйте `target/KillerEvent-1.0.0.jar` в папку `plugins/` вашего Paper сервера
2. Перезагрузите сервер командой `/reload confirm` или перезагрузите сервер

## Проверка установки

После загрузки сервера в логах должно появиться:
```
[KillerEvent] KillerEvent plugin enabled!
```

Плагин готов к использованию!
